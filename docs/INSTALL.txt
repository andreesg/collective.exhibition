collective.exhibition Installation
---------------------------------

To install collective.exhibition using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.exhibition`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.exhibition
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
